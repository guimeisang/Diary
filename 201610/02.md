### Function 类型

- 没有什么比上栗子来得理解深刻
```
function sum(num1,num2){
    return num1 + num2;
}
alert(sum(10,10)); //20

var anotherSum = sum;
alert(anotherSum(10,10)); //20

sum = null;
alert()
alert(anotherSum(10,10)); //20
```
**注意：使用不带圆括号的函数名是访问函数指针，而非调用函数，所以说anthoerSum和sum都指向同一个函数。**
